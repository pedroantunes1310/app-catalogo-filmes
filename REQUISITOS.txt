========PROJETO CATALOGO FILME========


REQUISITOS FUNCIONAIS:

CADASTRO:
- LOGIN DE USUARIO
- CADASTRO NOVO USUARIO
- LOGOUT

FILMES: 
- VISUALIZACAO DE FILMES 
- EXIBIR DETALHES DO FILME
- ICONE ALTERAR FILMES QUANDO EXPANDIDO CARD
- CADASTRAR NOVOS FILMES MODAL
- ALTERAR FILMES EXISTENTES
- FILTRAR/ORDENAR FILMES (BUSCAR, NOTA, ULTIMA ALTERACAO, STATUS)

DASHBOARD:
- ABA DASHBOARDS 
- CARREGAR DASHBOARDS ILUSTRATIVOS 
- EXIBIR GRAFICOS DE ACORDO COM SELEÇÃO
- EXIBIR FILMES TOTAIS ASSISTIDOS
- EXIBIR FILMES ASSISTIDOS NO MES
- EXIBIR MEDIA DE AVALIAÇÕES (NOTA)

REQUISITOS NAO FUNCIONAIS

CADASTRO:
- TODA TRANSACAO COM PERSISTENCIA EM DADOS DEVERÁ SER REALIZADA UTILIZANDO THREADS (DE MANEIRA ASSÍNCRONA)
- AUTENTICACAO COM GOOGLE (PERSISTIR FIREBASE)
- AUTENTICACAO EMAIL/SENHA (PERSISTIR FIREBASE, AUTENTICAR SE PARAMETROS CORRETOS, RECUSAR LOGIN SE PAREMETROS INCORRETOS)
- CADASTRO NOVO USUARIO (PERSISTIR FIREBASE, VALIDAÇÕES DE CAMPOS OBRIGATÓRIOS)
- SEGURANÇA DOS DADOS TRANSACIONADOS 

FILMES:
- BOTÃO CADASTRAR NOVO FILME ABRIR NOVA TELA/MODAL
- MODAL/TELA CADASTRAR NOVO FILM (VALIDAÇÕES DE CAMPO)
- BOTÃO SALVAR/SUBMIT LEVAR DADOS AO BANCO 
- PERSISTENCIA DE DADOS FIREBASE CADASTRO FILMES (THREAD ASSINCRONO) 
- VISUALIZAÇÃO DE CARD BUSCANDO ALL (CARREGANDO THREAD)
- DEMONSTRAR CAMPOS PRINCIPAIS NO CARD (TITULO, NOTA, STATUS)
- VISUALIZAR/OCULTAR DETALHES DO CARD (APRESENTAR DESCRIÇÃO/REVIEW, DATA ALTERAÇÃO, ICONE PARA EDIÇÃO)
- EDITAR FILMES PERMITINDO APENAS OS CAMPOS NAO OBRIGATORIOS DEFINIDOS NA ENTIDADE
- BOTÃO SALVAR/SUBMIT ATUALIZAR DADOS DO BANCO 

DASHBOARD:
- CARREGAR DASHS CALCULADOS (THREAD)
- DASH FILMES ASSISTIDOS MES (SOMA MES ATUAL)
- DASH FILMES ASSISTIDOS TOTAL (SOMA)
- DASH MEDIA DE AVALIAÇÕES (AVG NOTAS)
